⚙️ Liên Kết Real-Time Google Sheet & Lead Analytics

Trang web của bạn đang sử dụng hệ thống giả lập cục bộ (`localStorage`). Để chuyển sang **tự động hóa 100% bằng Google Sheets** và ghi nhận danh sách email khách hàng trên trang tính trực tuyến thực tế, hãy thực hiện theo các bước sau:

Hướng Dẫn Cấu Hình Apps Script Đơn Giản:

  1. Truy cập Google Sheets và tạo một bảng tính trống mới để chứa data khách hàng.
  2. Trên thanh công cụ phía trên, click chọn Tiện ích mở rộng (Extensions) -> Apps Script.
  3. Xóa toàn bộ mã code mặc định hiển thị trong trình soạn thảo, sao chép và dán đoạn mã code dưới đây vào:
function doPost(e) { try { var data = JSON.parse(e.postData.contents); var email = data.email; var app = data.app; var version = data.version; var lang = data.lang || 'vi'; var timestamp = new Date(); // Ghi nhận thông tin đăng ký tải xuống vào Google Sheet active var sheet = SpreadsheetApp.getActiveSpreadsheet().getActiveSheet(); sheet.appendRow([timestamp, email, app, version, lang]); return ContentService.createTextOutput(JSON.stringify({status: "success"})) .setMimeType(ContentService.MimeType.JSON); } catch (error) { return ContentService.createTextOutput(JSON.stringify({status: "error", message: error.toString()})) .setMimeType(ContentService.MimeType.JSON); } }
  1. Nhấn biểu tượng Lưu hình đĩa mềm trên thanh công cụ.
  2. Click vào nút Triển khai (Deploy) ở góc trên bên phải -> chọn Triển khai mới (New deployment).
  3. Ở bảng mở ra, click biểu tượng răng cưa chọn loại cấu hình là Ứng dụng web (Web app).
  4. Thiết lập:
    • Thực thi dưới dạng (Execute as): Chọn Tôi (Me).
    • Ai có quyền truy cập (Who has access): Chọn Bất kỳ ai (Anyone).
  5. Click Triển khai (Deploy), cấp quyền truy cập email và tài khoản của bạn, sau đó copy URL Ứng dụng web (Web app URL) và dán vào ô cấu hình phía trên của trang này!
