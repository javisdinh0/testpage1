HƯỚNG DẪN SỬ DỤNG USER GUIDE

ViD Block Library v1.0

I. Hướng dẫn sử dụng phần mềm

01

Khởi động công cụ

Trong môi trường AutoCAD, bạn gõ lệnh BI và nhấn Enter. Giao diện quản lý thư viện sẽ xuất hiện phía trên màn hình làm việc của bạn.

02

Tìm kiếm và Duyệt Block

Sử dụng bảng Tree View bên trái để tìm đến thư mục chứa cấu kiện bạn cần. Danh sách các Block sẽ hiện ra ở bảng bên cạnh.

03

Kiểm tra Preview

Khi nhấn vào tên Block, hình ảnh minh họa sẽ hiện lên ở khung Preview. Điều này giúp bạn xác nhận đúng cấu kiện trước khi chèn.

04

Thiết lập tỷ lệ và Chèn

Nhập giá trị vào ô Scale (ví dụ: 100 cho tỷ lệ 1/100). Sau đó Double-click vào ảnh Preview và chọn điểm đặt trên bản vẽ.

Lưu ý: Phần mềm hỗ trợ tự động Explode (phá khối) nếu bạn tích chọn vào ô "Auto Explode" trước khi chèn.

II. Hướng dẫn tự tạo thư viện riêng

Để xây dựng kho dữ liệu cá nhân, hãy làm theo các bước sau:

1. Chuẩn bị thư mục dữ liệu

Tìm đến thư mục cài đặt phần mềm, bạn sẽ thấy thư mục Data. Bạn có thể tạo thêm các thư mục con bên trong để phân loại.

2. Quy tắc File

Với mỗi cấu kiện, bạn cần 2 file có tên giống hệt nhau:

  • File bản vẽ: Định dạng .dwg.
  • File hình ảnh: Định dạng .png hoặc .jpg.

I. Software Usage Guide

01

Start the Tool

In the AutoCAD environment, type BI and press Enter. The library management interface will appear on your workspace.

02

Search and Browse Blocks

Use the Tree View panel on the left to navigate to the folder containing the components you need. The list of Blocks will appear in the adjacent panel.

03

Check Preview

When you click on a Block name, an illustration will appear in the Preview frame. This helps you confirm the correct component before insertion.

04

Set Scale and Insert

Enter a value in the Scale box (e.g., 100 for 1/100 scale). Then Double-click the Preview image and pick an insertion point in AutoCAD.

Note: The software supports automatic Explode if you check the "Auto Explode" box before insertion.

II. Creating Your Own Library

To build your personal database, follow these steps:

1. Prepare Data Folders

Locate the software installation directory and find the Data folder. You can create subfolders inside to categorize your blocks.

2. File Naming Rules

For each component, you need 2 files with the exact same name:

  • Drawing File: .dwg format.
  • Image File: .png or .jpg format.
Data/
└── 01_Steel_Sections/
├── I_Section.dwg
└── I_Section.png <-- MUST HAVE THE SAME NAME
