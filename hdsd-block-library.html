HƯỚNG DẪN SỬ DỤNG USER GUIDE

BLOCKMASTER | Powered by ViDLab

I. Quy trình thao tác phần mềm

01

Khởi động công cụ

Trong môi trường AutoCAD, gõ lệnh BLOCKMASTER và nhấn Enter để hiển thị thanh điều hướng quản lý thư viện Block ở cạnh phải màn hình.

02

Tìm kiếm & Duyệt thư mục cây

Duyệt sơ đồ danh mục cây (Tree View) lồng trực tiếp dưới tab "My Library". Các accordion sẽ trượt lên xuống mượt mà giúp bạn gom gọn cấu kiện.

03

Lưu block nhanh với "Get From Text"

Nhấp ADD TO LIBRARY $\rightarrow$ Click GET FROM TEXT để nhấp chọn chữ ghi chú trên bản vẽ làm tên tệp nhanh chóng mà không cần nhập tay.

04

Chèn block kèm scale dòng lệnh

Nhấp đúp vào block cần chèn, tại dòng lệnh AutoCAD nhập tỷ lệ thu nhỏ (ví dụ: nhập 10 để thu nhỏ 1/10), sau đó chọn vị trí điểm đặt trên bản vẽ.

05

Dọn dẹp thư viện an toàn

Để xóa tệp lỗi thời, click chọn block và nhấn nút phẳng đỏ 🗑 DELETE BLOCK. Hệ thống giải phóng bộ nhớ GDI+ và xóa sạch DWG/Ảnh xem trước vật lý trên ổ đĩa lập tức.

06

Chỉnh sửa Block trực tiếp

Chọn block và nhấn nút ✏️ EDIT BLOCK. BLOCKMASTER tự động tìm kiếm và mở trực tiếp tệp bản vẽ nguồn .dwg trong AutoCAD để bạn chỉnh sửa ngay lập tức.

II. Hướng dẫn thiết lập cơ sở dữ liệu riêng

Để tích hợp thư viện thiết kế riêng của công ty, hãy tuân thủ cấu trúc thư mục sau:

1. Tổ chức phân cấp thư mục

Tạo các thư mục con trong thư mục Data gốc của phần mềm. Sơ đồ cây của BLOCKMASTER sẽ tự động đồng bộ hóa cấu trúc này.

2. Quy tắc đặt tên tệp đồng bộ

Mỗi cấu kiện bắt buộc phải có 2 tệp trùng tên nằm trong cùng thư mục:

  • Tệp bản vẽ: .dwg (chứa thực thể block định nghĩa).
  • Tệp ảnh xem trước: .png hoặc .jpg (ảnh minh họa kích thước chuẩn 256x170).

I. Software Operation Workflow

01

Launch the Plugin

Within AutoCAD, type BLOCKMASTER and hit Enter to display the elegant sidebar manager on the right side of the screen.

02

Browse Nested Tree Categories

Navigate the folder Tree View placed inside the "My Library" button. Accordion transitions slide smoothly to conserve vertical screen space.

03

Add Blocks with "Get From Text"

Click **ADD TO LIBRARY** $\rightarrow$ Press **GET FROM TEXT** to pick database texts directly on the CAD view. The extracted cleaned string automatically names the block.

04

Insert with Command-Line Scaling

Double-click the block, type the scale denominator factor in AutoCAD command bar (e.g. type 10 for 1/10 scale-down), then pick the insertion coordinate point.

05

Physically Delete Unused Assets

Click a block card and hit 🗑 DELETE BLOCK. The tool disposes file handles and completely removes DWG drawings and PNG/JPG previews from your disk.

06

Direct Block Editing

Select any block and click ✏️ EDIT BLOCK. BLOCKMASTER instantly opens the source .dwg drawing in the active AutoCAD session for direct modification.

II. Setting Up Custom Databases

To integrate custom company blocks, follow these layout specifications:

1. Hierarchical Directory Hierarchy

Create subfolders under the main Data directory. BLOCKMASTER dynamically constructs the tree based on these real directories.

2. Synchronized Naming Strategy

Each block component must comprise 2 files named identically inside the same subfolder:

  • Drawing File: .dwg (contains the geometric block block definitions).
  • Thumbnail Image: .png or .jpg (dimensions recommended: 256x170).
Data/
├── 01_Foundation_Layouts/
│ ├── Footing_F1.dwg
│ └── Footing_F1.png <-- MUST BE INDENTICAL FILE NAME
└── 02_Column_Details/
├── Column_C1.dwg
└── Column_C1.png
