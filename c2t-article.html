C2T (Cad to Tekla) is one of the most powerful tools in iViDLab's solution suite, designed to eliminate the barrier between AutoCAD and Tekla Structures during the project setup phase.

The Structural Engineer's Pain Point

Typically, when starting a Tekla project, an engineer must manually enter dozens of X, Y, Z coordinates from AutoCAD architectural/structural drawings. This is not only time-consuming but also extremely error-prone, leading to discrepancies in the entire 3D model later on.

"C2T allows you to simply SELECT dimensions in AutoCAD, and the tool automatically calculates, sorts, and exports an accurate Grid (.grd) file for Tekla."

Core Features

  • Extract Dimensions: Automatically retrieves values from Dim lines in AutoCAD, sorted in ascending order.
  • Sequence Labeling: Automatically generates grid labels (A, B, C... or 1, 2, 3...) with customizable prefixes.
  • Z-Level Calculation: Supports entering a Base Level and automatically accumulates distances to calculate accurate floor elevations.
  • Direct Export: Exports the .grd file directly into the currently open Tekla model folder.

Simple Usage

  1. Open the plan drawing in AutoCAD.
  2. Run the C2T command to open the tool interface.
  3. Select Dim lines along the X and Y directions.
  4. Press 'Export' and return to Tekla to update the Grid.

Technology Stack

The tool is written in C#, utilizing the AutoCAD .NET API and Tekla Open API to ensure seamless interaction and maximum performance.
