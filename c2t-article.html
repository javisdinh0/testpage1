C2T: Giải Pháp Đồng Bộ Lưới Trục Từ AutoCAD Sang Tekla C2T: Grid Synchronization Solution from AutoCAD to Tekla

Chuyên mục: CAD To Tekla AutomationCategory: CAD To Tekla Automation • Ngày đăng: 16/05/2026Published: May 16, 2026 • Tác giả: Admin iViDLabAuthor: Admin iViDLab

C2T (Cad to Tekla) là một trong những công cụ mạnh mẽ nhất trong bộ giải pháp của iViDLab, được thiết kế để xóa bỏ rào cản giữa AutoCAD và Tekla Structures trong giai đoạn thiết lập dự án.

Nỗi đau của kỹ sư kết cấu

Thông thường, khi bắt đầu một dự án Tekla, kỹ sư phải nhập tay hàng chục tọa độ X, Y, Z từ bản vẽ kiến trúc/kết cấu AutoCAD. Việc này không chỉ tốn thời gian mà còn cực kỳ dễ nhầm lẫn, dẫn đến sai lệch toàn bộ mô hình 3D sau này.

"C2T cho phép bạn chỉ cần QUÉT CHỌN các Dimension trên AutoCAD, và công cụ sẽ tự động tính toán, sắp xếp và xuất ra file Grid (.grd) chuẩn xác cho Tekla."

Các tính năng cốt lõi

  • Extract Dimensions: Tự động lấy giá trị từ các đường Dim trên AutoCAD, sắp xếp theo thứ tự tăng dần.
  • Sequence Labeling: Tự động sinh chuỗi tên trục (A, B, C... hoặc 1, 2, 3...) với các tiền tố tùy chỉnh.
  • Z-Level Calculation: Hỗ trợ nhập Base Level và tự động cộng dồn các khoảng cách để tính cao độ sàn chính xác.
  • Direct Export: Xuất trực tiếp file .grd vào thư mục model Tekla đang mở.

Cách sử dụng đơn giản

  1. Mở bản vẽ mặt bằng trên AutoCAD.
  2. Chạy lệnh C2T để mở giao diện công cụ.
  3. Quét chọn các đường Dim theo phương X và Y.
  4. Nhấn 'Export' và quay lại Tekla để cập nhật lưới trục (Grid).

Công nghệ sử dụng

Công cụ được viết bằng ngôn ngữ C#, sử dụng AutoCAD .NET API và Tekla Open API để đảm bảo sự tương tác mượt mà và hiệu suất cao nhất.

C2T (Cad to Tekla) is one of the most powerful tools in iViDLab's solution suite, designed to eliminate the barrier between AutoCAD and Tekla Structures during the project setup phase.

The Structural Engineer's Pain Point

Typically, when starting a Tekla project, an engineer must manually enter dozens of X, Y, Z coordinates from AutoCAD architectural/structural drawings. This is not only time-consuming but also extremely error-prone, leading to discrepancies in the entire 3D model later on.

"C2T allows you to simply SELECT dimensions in AutoCAD, and the tool automatically calculates, sorts, and exports an accurate Grid (.grd) file for Tekla."

Core Features

  • Extract Dimensions: Automatically retrieves values from Dim lines in AutoCAD, sorted in ascending order.
  • Sequence Labeling: Automatically generates grid labels (A, B, C... or 1, 2, 3...) with customizable prefixes.
  • Z-Level Calculation: Supports entering a Base Level and automatically accumulates distances to calculate accurate floor elevations.
  • Direct Export: Exports the .grd file directly into the currently open Tekla model folder.

Simple Usage

  1. Open the plan drawing in AutoCAD.
  2. Run the C2T command to open the tool interface.
  3. Select Dim lines along the X and Y directions.
  4. Press 'Export' and return to Tekla to update the Grid.

Technology Stack

The tool is written in C#, utilizing the AutoCAD .NET API and Tekla Open API to ensure seamless interaction and maximum performance.
