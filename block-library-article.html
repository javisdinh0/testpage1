ViD Block Library: Hệ Thống Quản Lý Thư Viện CAD Thông Minh ViD Block Library: Intelligent CAD Library Management System

Chuyên mục: CAD Productivity ToolsCategory: CAD Productivity Tools • Ngày đăng: 16/05/2026Published: May 16, 2026 • Tác giả: Admin ViDLabAuthor: Admin ViDLab

Trong công việc triển khai bản vẽ (Detailing), việc quản lý và tìm kiếm hàng ngàn Block là một cực hình. ViD Block Library được xây dựng để biến thư viện hỗn độn của bạn thành một kho tàng có tổ chức, trực quan và dễ sử dụng.

Quản lý bằng hình ảnh trực quan

Thay vì phải nhớ tên file khô khan (ví dụ: COL_300x500_SEC.dwg), ViD Block Library tự động nhận diện các file ảnh minh họa cùng tên trong thư mục. Bạn chỉ cần rê chuột qua danh sách để xem trước (Preview) hình ảnh thực tế của Block trước khi chèn vào bản vẽ.

"Không cần phải mở từng file để kiểm tra nội dung. Với ViD Block Library, bạn nhìn thấy những gì bạn sắp chèn."

Các tính năng chính

  • Tree View Folder: Tổ chức thư viện theo cấu trúc thư mục phân cấp, giúp quản lý hàng ngàn block theo từng hạng mục.
  • Auto Preview: Hiển thị hình ảnh thu nhỏ cực nhanh, hỗ trợ định dạng PNG, JPG, JPEG.
  • Smart Scale & Insert: Tùy chọn chèn block vào điểm bất kỳ, hỗ trợ tính toán tỷ lệ scale tự động phù hợp với khung bản vẽ hiện hành.
  • Auto Explode: Tùy chọn tự động phá khối sau khi chèn để chỉnh sửa ngay lập tức.

Hướng dẫn sử dụng nhanh

Để bắt đầu sử dụng ViD Block Library, bạn chỉ cần thực hiện các bước đơn giản sau:

  • Mở công cụ: Gõ lệnh BI trong AutoCAD để gọi giao diện quản lý.
  • Duyệt danh mục: Sử dụng cây thư mục bên trái để di chuyển giữa các nhóm Block.
  • Xem trước: Nhấn vào tên Block trong danh sách để xem hình ảnh minh họa.
  • Chèn Block: Nhấn đúp (Double-click) vào hình ảnh preview hoặc nhấn nút "Insert".
Mẹo: Bạn có thể nhập tỷ lệ Scale trước khi chèn để Block tự động khớp với tỷ lệ bản vẽ hiện hành (ví dụ: nhập 100 để chèn block tỷ lệ 1/100).

In detailing work, managing and searching through thousands of Blocks can be a nightmare. ViD Block Library is built to turn your messy library into an organized, visual, and easy-to-use treasure trove.

Visual Image Management

Instead of having to remember dry filenames (e.g., COL_300x500_SEC.dwg), ViD Block Library automatically recognizes image files with the same name in the folder. You just need to hover over the list to Preview the actual image of the Block before inserting it into the drawing.

"No need to open every file to check content. With ViD Block Library, you see what you're about to insert."

Key Features

  • Tree View Folder: Organizes the library in a hierarchical folder structure, helping manage thousands of blocks by category.
  • Auto Preview: Displays thumbnails extremely fast, supporting PNG, JPG, and JPEG formats.
  • Smart Scale & Insert: Options to insert blocks at any point, supporting automatic scale calculation to fit the current drawing frame.
  • Auto Explode: Optional automatic block exploding after insertion for immediate editing.

Quick Start Guide

To start using ViD Block Library, simply follow these easy steps:

  • Open tool: Type BI in AutoCAD to bring up the management interface.
  • Browse categories: Use the folder tree on the left to move between Block groups.
  • Preview: Click on a Block name in the list to view its illustration.
  • Insert Block: Double-click the preview image or press the "Insert" button.
Tip: You can enter a Scale ratio before insertion so the Block automatically matches the current drawing scale (e.g., enter 100 to insert a 1/100 scale block).
