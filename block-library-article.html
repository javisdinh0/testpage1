Trong công việc thiết kế và triển khai bản vẽ kỹ thuật, việc quản lý hàng ngàn Block mẫu luôn là bài toán khó khăn. BLOCKMASTER ra đời dưới sự phát triển của iViDLab nhằm mang tới một giải pháp quản lý thư viện Block thông minh, trực quan và tối ưu hiệu suất làm việc vượt trội.

Giao diện hiện đại & Xem trước trực quan

Với phong cách thiết kế GitHub Dark cao cấp, BLOCKMASTER mang lại trải nghiệm làm việc mượt mà, hạn chế mỏi mắt. Bạn không cần phải mở từng file để kiểm tra nội dung; hình ảnh minh họa sẽ tự động tải lên ngay lập tức giúp bạn duyệt tìm dễ dàng.

"Không còn những dòng tên khô khan. BLOCKMASTER trực quan hóa toàn bộ thư viện Block của bạn ngay trên bản vẽ."

Các tính năng đột phá mới

  • Sơ đồ cây lồng sidebar (Tree View Accordion): Cây thư mục được lồng trực tiếp trong Sidebar trái giúp thu gọn/mở rộng accordion trượt cực kỳ mượt mà.
  • Bilingual dynamic switcher: Hỗ trợ chuyển đổi nhanh ngôn ngữ tiếng Anh và tiếng Việt trực tiếp trên giao diện Ribbon và bảng điều khiển WinForms Palette.
  • Nhập tỷ lệ scale ngược dòng lệnh: Trước khi chèn, phần mềm nhắc bạn nhập tỷ lệ từ dòng lệnh (ví dụ nhập 10 để tự động thu nhỏ 1/10), áp dụng hoàn hảo cho cả chèn thường và chèn rã khối (Explode Insert).
  • Trích xuất chữ làm tên block (Get From Text): Tạo block mới siêu nhanh bằng cách chọn trực tiếp text trên bản vẽ làm tên file, tự động lọc sạch ký tự đặc biệt.
  • Trình xóa tệp an toàn (Anti-Lock Delete): Nút xóa đỏ phẳng sang trọng cho phép xóa tệp bản vẽ và ảnh xem trước vật lý trên ổ cứng an toàn mà không bị Windows khóa tệp.
  • Chỉnh sửa nguồn trực tiếp (Edit Block): Mở trực tiếp tệp nguồn DWG trong AutoCAD để chỉnh sửa ngay lập tức mà không cần tìm kiếm file thủ công.

In drafting and detailing workflows, managing thousands of drawing Blocks can be highly repetitive and time-consuming. BLOCKMASTER is engineered by iViDLab to provide an intelligent, visual, and highly efficient solution to govern your entire CAD block repository.

Modern Dark UX & Visual Previews

Following a premium GitHub Dark aesthetic, BLOCKMASTER ensures comfortable drafting during extended hours. You no longer need to remember obscure file names or manually open files to verify contents; accurate image thumbnails are rendered instantly.

"No more cryptic file names. BLOCKMASTER visualizes your entire drawing asset library right inside AutoCAD."

Breakthrough Features

  • Nested Sidebar Tree Accordion: The directory folder explorer is embedded directly under the My Library tab, expanding with responsive accordion animations.
  • Dynamic English/Vietnamese Toggle: Access prompts, forms, and alerts instantly in your preferred language without restarts.
  • Command-line Division Scale Prompt: Prompts you natively via AutoCAD's command bar for a division factor (e.g., entering 10 scales the block down to exactly 1/10 or 0.1 scale). Supports both standard and explode block insertions.
  • Smart Text Extractor (Get From Text): Instantly extracts block names by selecting single-line (DBText) or multi-line (MText) objects on-screen, cleaning invalid path characters automatically.
  • Anti-Lock Physical File Deletion (🗑 DELETE BLOCK): A sleek red button that disposes GDI+ preview handles and cleans both the DWG and PNG/JPG files safely from disk.
  • Direct Source Editing (✏️ EDIT BLOCK): Instantly opens the source DWG file inside the active AutoCAD workspace for direct drafting and modification.
